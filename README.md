# SleepGuard

Windows の自動スリープを指定プロセスが動いている間だけ抑止するシステムトレイツールです。

## 動作原理

10秒ごとに監視対象プロセスを確認し、動作中であれば Windows のアイドルタイマーをリセットします。プロセスが消えたら何もしないのでWindows のスリープ設定に従って自然にスリープに入ります。

| 状態 | 動作 |
|------|------|
| プロセスあり（前回もあり） | アイドルタイマーリセット継続 |
| プロセスあり（前回なし） | アイドルタイマーリセット開始 |
| プロセスなし（前回もなし） | 何もしない、Windowsタイマーが進む |
| プロセスなし（前回あり） | リセット停止、Windowsタイマーが進む |

## 主な機能

- 監視対象プロセスを複数登録（EXEパス指定で同名プロセスを区別）
- 実行中プロセス一覧から直接追加
- スリープ猶予時間（0〜60分）
- ディスプレイのスリープも防止するオプション
- タスクトレイ常駐、Windows起動時に自動起動
- 動作ログ記録（EXEと同フォルダに `sleepguard.log`）

## 動作環境

- Windows 10 / 11 (x64)
- .NET 8 Runtime（要インストール）

## インストール

1. [Releases](https://github.com/morannbon/SleepGuard/releases) から最新の ZIP をダウンロード
2. 任意のフォルダに展開
3. `SleepGuard.exe` を実行

## ビルド

- Visual Studio 2022
- .NET 8 SDK
- `SleepGuard.sln` を開いて Release x64 でビルド
